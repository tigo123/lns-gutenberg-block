== Description ==

Gutenberg Blocks used for your website. Developed by LeNamSite

== Installation ==

1. Put source code into a folder
2. Open Terminal at source code folder
3. Run command: npm i
4. Run command: npm run build-all
5. Run command: npm run plugin-zip
6. Install plugin via .zip file
7. Active plugin
